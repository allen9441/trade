# AI Quant Trading Prediction

基於 LSTM 深度學習模型的台灣股市量化交易預測系統。

## 功能特色
- 支援多檔台灣上市櫃股票及 ETF 的歷史資料抓取 (包含 0050, 0056, 台積電, 鴻海 等權值股)。
- 計算多種技術指標（SMA、RSI、MACD、布林通道等）。
- 整合台灣加權指數（TWII）和 S&P 500 等大盤特徵。
- 使用 LSTM 模型進行多資產的時間序列預測。
- 定期重新訓練或載入已訓練的模型權重。
- 每日生成最新交易訊號（BUY / SELL），並依據模型信心度排序匯出成 JSON 格式供其他系統介接。

## 系統架構
- `main.py`: 系統主程式，負責資料抓取、模型訓練及訊號生成。
- `data/loader.py`: 處理股票與大盤數據下載，以及技術指標的計算。
- `models/lstm.py`: 定義及實作 LSTM 交易預測模型。
- `test.py`: 模型回測介面。
- `multi.py`: 同樣是回測，但可執行複數年份。
- `veri.py`: 驗證交易log。

## 安裝與執行

### 環境需求
- Python 3.8+
- 建議使用虛擬環境（推薦使用uv）

### 安裝依賴
請確保已安裝 `requirements.txt` 中的所有套件：
```bash
uv pip install -r requirements.txt
```

### 執行預測
執行主程式以生成最新的交易訊號：
```bash
python main.py
```

執行完成後，系統會在根目錄生成 `latest_signals.json`，裡面包含最新一個交易日的建議操作與模型信心度。
